### Modos de inicio

- CLI
npm run cli help

- Modo developer FrontEnd mas BackEnd
npm run dev 

- Modo developer FrontEnd
npm run f 

- Modo developer BackEnd
npm run b

- Migracion de la BD
npm run mig

- Migracion del tema css
npm run theme

- Script de checkeo de codigo
npm run check
